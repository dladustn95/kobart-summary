# kobart-summary

## Install KoBART
```
pip install git+https://github.com/SKT-AI/KoBART#egg=kobart
```

## Download binary
```
pip install gdown
python download_binary.py

kobart_summary
├── config.json
├── pytorch_model.bin
```
## Requirements
```
pytorch==1.9.0
transformers==4.8.2
pytorch-lightning==1.3.8
streamlit==0.72.0
```
